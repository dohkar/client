# Dohkar

Этот репозиторий содержит базовое приложение на [Next.js](https://nextjs.org/).

## Начало работы

1. Установите зависимости:

   ```
   npm install
   ```

2. Создайте файл переменных окружения `.env.local` при необходимости.

3. Запустите приложение в режиме разработки:

   ```
   npm run dev
   ```

4. Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Доступные скрипты

- `npm run dev` — запуск в режиме разработки
- `npm run build` — создание production-сборки
- `npm run start` — запуск production-сервера
- `npm run lint` — проверка кода линтером
- `npm run format` — форматирование кода с помощью Prettier
- `npm run format:check` — проверка форматирования без изменений

## Структура проекта

```
├── app/         # роутинг и страницы (Next.js App Router)
├── src/         # исходный код, компоненты, хуки, сервисы и др.
├── public/      # статические файлы
```
